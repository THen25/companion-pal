# Companion Pal 🐾

A responsive single-page website for a fictional pet adoption service, built as one of my first independent front-end projects.

## Live Demo

https://then25.github.io/companion-pal/

## About the Project

Companion Pal is a landing page for a fictional pet adoption service. It was one of the first sites I built completely on my own — no tutorials guiding me step by step. The goal was to put my HTML and CSS skills together into a real, cohesive project from scratch.

## Features

- Responsive layout that works on mobile, tablet, and desktop
- Sticky navigation bar with a mobile hamburger menu
- Full-width hero section with overlay text
- Mission statement section
- Responsive CSS Grid photo gallery of adoptable animals
- Contact form

## Built With

- HTML5
- CSS3 (Flexbox, Grid)
